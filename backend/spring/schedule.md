## Schedule

### Cron

```sh
* * * * * *
# 초(0-59) 분(0-59) 시(0-23) 일(1-31) 월(1-12 or JAN-DEC) 요일(0-7 or MON-SUN)
```

- `*` : 모든 값에 실행
- 숫자가 들어가면 각 항목이 그 값과 일치하면 실행
- `/` : `*/10 * * * * *` 10분마다 실행 (0, 10, 20...)
- range `-` : `0 0 8-10 * * *` 8~10시 정각마다 실행
- `,` : `0 0 6,19 * * *` 6시, 19시 정각마다 실행
- `?` : 해당 없음, 알 수 없음. `0 0 0 25 12 ?` 12월 25일 자정에 실행 (`0 0 0 25 12 *` 사용해도됨)
