## Next.js

SEO를 위한 SSR을 가능하게 해주는 프레임워크

### SEO

Search Engine Optimization
검색엔진들이 서버에 등록된 웹사이트의 html문서를 분석하여, 사이트를 검색 결과에 더 잘 노출될 수 있도록 함

### SSR

Server-Side Rendering
서버로부터 완전하게 만들어진 html을 받아와 페이지 전체를 렌더링하는 것\

### 장점

- 사전 렌더링, 서버사이드 렌더링
- Hot Code Reloading
