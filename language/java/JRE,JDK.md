## JDK, JRE

### JDK (Java Development Kit)

- Java를 사용하기 위해 필요한 모든 기능을 갖춘 Java용 SDK (Software Development Kit)
- JDK에는 JRE가 포함되어 있음
- Java 프로그램을 **생성, 실행, 컴파일**할 수 있음

### JRE (Java Runtime Environment)

- JVM + Java Class Library등으로 구성
- 컴파일된 Java 프로그램을 **실행**하는데 필요한 패키지
