## Object class

- java.lang.Object
- 모든 자바 클래스의 조상 클래스
- 그래서 클래스들은 Object 클래스의 메서드를 사용할 수 있음
- 필드는 없고, 메서드만 있음
- Object클래스를 타입으로 설정하면, 모든 클래스의 상위 클래스이기때문에 모든 값이 들어올 수 있음 (any와 유사)

### 메서드

- equals()
- toString()
- clone()
- hasgCode()
- 등, 총 11개의 메서드가 있음
